# 关于开展学校各二级网站检查的通知

各单位：

为持续提升学校各单位网站建设管理水平，2021年12月，党委宣传部对学校各单位网站进行了检查并对检查结果进行了通报。为落实各单位网站整改情况，规范校园网站管理与信息发布，充分发挥校园网站在教学、科研、管理、党建、宣传和思想政治教育工作中的作用，学校决定再次对二级网站进行检查。现将有关事项通知如下:

一、**检查范围**：

全校二级网站。

二、**检查时间**

（一）自查完善阶段：4月15日－5月6日，各单位依据《关于学校各二级网站检查情况的通报》（附表1）对本单位网站开展自查，及时更新、整改。

（二）组织检查阶段：5月7日开始，党委宣传部、现代教育技术中心组成检查小组对各单位网站进行检查，并对检查结果进行通报。

三、**检查内容**

检查内容主要围绕网站管理、栏目设置、安全运行、内容更新等四个方面进行检查。具体要求如下：

1.网站管理。重点检查是否有单位领导分管网站工作，是否指定专人管理网站并进行实际管理，是否指定专人发布网站信息。

2.栏目设置。重点检查网站栏目是否合理，用户获取信息是否便捷有效等。

3.安全运行。重点检查信息系统（网站）的弱密码、更新率、使用率和非法链接、安全漏洞、不良信息，外链网站是否安全；是否做到“敏感、不良、涉密信息不上网，上网信息准确、脱敏、不涉密”。

4.内容更新。各二级网站栏目内容是否及时更新，是否存在服务信息实际不能获取、错误引导、链接打不开、服务内容不存在、数据滞后、内容陈旧等问题。

四、相关要求

1.全校各单位于5月6日前做好本单位网站建设和更新工作，对陈旧信息进行及时更新，重点检查网站是否存在无内容的“死链接”或链接错误现象。

2.请各单位认真填写《湖南工业大学二级网站自查工作报表》（附件2），于5月6日17:00前把纸质版报送党委宣传部414室，电子版发送至宣传部办公邮箱（xcb@hut.edu.cn）。

3.检查结果将全校通报。
